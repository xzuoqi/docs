# EDL 按键

瑞莎 Dragon Q6A 板载 EDL 按键，用于进入 EDL 模式，EDL 模式是高通处理器设备中的一种紧急下载模式，主要用于紧急修复、刷写固件或解锁设备。

## 使用指南

主板上电的过程中检测到 EDL 按键被按下，主板将进入 EDL 模式。

详细步骤：

1. 按住 EDL 按键
2. 连接 12V Type-C 电源适配器（兼容 PD 协议）
3. 松开 EDL 按键

正常情况下，主板会进入 EDL 模式。
