# Tina Linux 系统

## 系统概述

**Tina Linux** 是全志科技（Allwinner）专为其 SoC 平台开发的嵌入式 Linux 操作系统。基于 OpenWrt 构建，针对全志芯片平台进行了深度优化，为 Cubie A5E 提供了高效、稳定的嵌入式系统解决方案。
