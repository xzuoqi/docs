# 从 SD 卡安装系统到 NVMe

## 启动系统

完成系统的安装后，将主板关机，断开主板电源，按照以下操作完成系统启动：

① : 将 MicroSD 卡从主板上取下

② : 将 NVMe SSD 安装到主板上

③ : 使用 HDMI 数据线连接显示器和主板

④ : 使用 USB-A 接口连接鼠标或键盘：若有 USB-A 拓展坞，可以通过拓展坞连接鼠标和键盘

⑤ : 使用 5V Type-C 电源适配器给主板供电

若主板供电正常，主板电源指示灯绿色亮起；若系统启动正常，状态指示灯蓝色闪烁。

成功启动系统后，显示器会显示系统桌面。
