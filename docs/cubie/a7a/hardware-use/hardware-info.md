- 电源指示灯

Cubie A7A 正常供电，电源指示灯绿色亮起；断电或关机时，电源指示灯熄灭。

- 状态指示灯

系统正常启动时，状态指示灯蓝色闪烁；系统异常时，状态指示灯熄灭。

- LDDR5 内存

LDRR5 内存可选：2GB / 4GB / 8GB / 16GB。
