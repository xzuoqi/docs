此文档讲述如何使用 QAI AppBuilder python 库在瑞莎 Dragon Q6A 上运行 [YOLOv8-det](https://aihub.qualcomm.com/models/yolov8_det?domain=Computer+Vision&useCase=Object+Detection)
目标检测模型。

## 安装 QAI AppBuilder

:::tip
请根据 [**QAI AppBuilder 安装方法**](qai-appbuilder) 安装 QAI AppBuilder。
:::

## 运行示例

### 安装依赖

<NewCodeBlock tip="Device" type="device">

```bash
pip3 install requests tqdm qai-hub py3_wget opencv-python torch
```

</NewCodeBlock>

### 运行脚本

- 进入示例目录

   <NewCodeBlock tip="Device" type="device">

  ```bash
  cd ai-engine-direct-helper/samples/linux/python
  ```

    </NewCodeBlock>

- 准备输入图片，这里以以下图片为输入示例

  <div style={{ textAlign: "center" }}>
    <img src="/img/dragon/q6a/yolov8-det_input.webp" style={{ width: "50%" }} />
    input image
  </div>

- 执行推理

  <NewCodeBlock tip="Device" type="device">

  ```bash
  python3 yolov8_det/yolov8_det.py --input_image_path ./bus.jpg
  ```
