介绍进入 EDL 模式的方法。

## EDL 模式说明

高通 EDL（Qualcomm Download Mode）模式是高通处理器设备中的一种紧急下载模式，主要用于紧急修复、刷写固件或解锁设备。

## 进入 EDL 模式

你需要在主板上电前按住主板上的 `EDL` 按键，主板上电后，可以松开 `EDL` 按键，设备会自动进入 EDL 模式。
