## 启动系统

完成系统的安装后，将系统启动介质安装到主板对应位置，然后使用电源适配器给主板供电并启动系统。

:::tip 系统启动优先级
若您在主板上安装了多个系统启动介质，系统会默认从优先级较高的系统启动介质启动：若想要从低优先级的系统启动介质启动，可以将高优先级的系统启动介质取下

{props.system_start_priority}

:::

## 使用系统

主要给有屏和无屏模式的用户提供一些建议，方便大家更好的上手使用我们的主板。

### 有屏模式

有屏模式指的是使用 HDMI 数据线连接主板和显示器，通过显示器显示系统界面和使用鼠标和键盘进行图形化界面操作。

### 无屏模式

[无屏模式](../system-config/screenless)指的是不使用显示器，建议用户通过[串口登录](../system-config/uart_login)或[SSH 登录](../system-config/ssh_login)和控制系统。
