# DX Model Zoo

**DX Model Zoo** 包含一些列已经编译过的目标分类，目标识别，图像分割，面部识别， 图片降噪等 dxnn 模型。
每个模型都提供原模型来源，onnx 模型，编译后的 dxnn 模型，编译配置文件下载，并附有的 benchmark， accuracy 等数据。
