- Power Indicator

When the Cubie A7Z is powered on normally, the power indicator will light up green.

- LPDDR4/4x

LPDDR4/4x options available: 1GB / 2GB / 4GB / 8GB / 16GB

- Onboard UFS

The onboard UFS module supports up to 1TB of storage capacity.
