# Using PoE HAT

The Gigabit Ethernet port (near USB) on ROCK 4B/4B+/4SE supports Power over Ethernet (PoE) with the additional Radxa 23W PoE HAT.

:::tip

ROCK 4A/4A+ does not support PoE.

:::
