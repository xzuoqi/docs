This document explains how to enter EDL mode.

## About EDL Mode

Qualcomm EDL (Emergency Download Mode) is an emergency download mode in Qualcomm processor devices, primarily used for emergency repairs, firmware flashing, or device unlocking.

## Entering EDL Mode

To enter EDL mode, press and hold the `EDL` button on the board before powering it on. After the board is powered on, you can release the `EDL` button, and the device will automatically enter EDL mode.
