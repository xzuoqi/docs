This document explains how to use the QAI AppBuilder Python library to run the [Real-ESRGAN-x4plus](https://aihub.qualcomm.com/models/real_esrgan_x4plus?domain=Computer+Vision&useCase=Super+Resolution&chipsets=qualcomm-qcs6490-proxy)
image super-resolution model on the Radxa Dragon Q6A.

## Installing QAI AppBuilder

:::tip
Please install QAI AppBuilder according to the [**QAI AppBuilder Installation Guide**](qai-appbuilder).
:::

## Running the Example

### Installing Dependencies

<NewCodeBlock tip="Device" type="device">

```bash
pip3 install requests tqdm qai-hub py3_wget opencv-python torch
```

</NewCodeBlock>

### Running the Script

- Navigate to the example directory

   <NewCodeBlock tip="Device" type="device">

  ```bash
  cd ai-engine-direct-helper/samples/linux/python
  ```

    </NewCodeBlock>
