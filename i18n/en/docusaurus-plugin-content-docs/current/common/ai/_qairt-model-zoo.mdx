For common open-source AI models in various domains, we provide pre-compiled models optimized for different NPU hardware architectures that are ready for download.
Users can directly download these models and deploy them using [QAIRT](qairt-sdk#qairt) or [QAI-APPBUILDER](qai-appbuilder).

Model categories include:

- Computer Vision Models

- Speech Models

- Generative Models

- Multimodal Models
