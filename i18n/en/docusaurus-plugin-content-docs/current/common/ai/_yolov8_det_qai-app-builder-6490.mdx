This document explains how to use the QAI AppBuilder Python library to run the [YOLOv8-det](https://aihub.qualcomm.com/models/yolov8_det?domain=Computer+Vision&useCase=Object+Detection)
object detection model on the Radxa Dragon Q6A.

## Installing QAI AppBuilder

:::tip
Please install QAI AppBuilder according to the [**QAI AppBuilder Installation Guide**](qai-appbuilder).
:::

## Running the Example

### Installing Dependencies

<NewCodeBlock tip="Device" type="device">

```bash
pip3 install requests tqdm qai-hub py3_wget opencv-python torch
```

</NewCodeBlock>

### Running the Script

- Navigate to the example directory

   <NewCodeBlock tip="Device" type="device">

  ```bash
  cd ai-engine-direct-helper/samples/linux/python
  ```

    </NewCodeBlock>

- Prepare the input image. The following image is used as an example input

  <div style={{ textAlign: "center" }}>
    <img
      src="/en/img/dragon/q6a/yolov8-det_input.webp"
      style={{ width: "50%" }}
    />
    Input image
  </div>

- Run the inference

  <NewCodeBlock tip="Device" type="device">

  ```bash
  python3 yolov8_det/yolov8_det.py --input_image_path ./bus.jpg
  ```
