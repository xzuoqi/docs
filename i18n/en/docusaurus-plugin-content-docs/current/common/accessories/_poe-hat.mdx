## Fan Speed Control

### Enable PoE Overlay

To enable the temperature-controlled fan and temperature sensor, you need to enable the corresponding overlay.

Please refer to the [Device Tree Configuration](../../radxa-os/rsetup#overlays) and enable the overlay named "Enable Radxa {props.pwr_rate}W PoE HAT".

<pre style={{ whiteSpace: "nowrap" }}>
  [*] Enable Radxa {props.pwr_rate}W PoE HAT
</pre>

Finally, reboot the device to load the overlay.

:::info
If you cannot find the corresponding overlay in `rsetup`, please refer to the [System Update Guide](../../radxa-os/rsetup#system-update) to update your system first, and then perform the above operations.
:::
